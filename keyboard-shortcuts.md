## Keyboard Shortcuts

The modifier key is <kbd>Cmd</kbd> on macOS and <kbd>Ctrl</kbd> on Linux/Windows.

| Shortcut | Action |
|---|---|
| <kbd>Mod+N</kbd> | New request |
| <kbd>Mod+S</kbd> | Save request |
| <kbd>Mod+Enter</kbd> | Send request |
| <kbd>Mod+W</kbd> | Close tab |
| <kbd>Mod+B</kbd> | Toggle sidebar |
| <kbd>Mod+,</kbd> | Open settings |
| <kbd>Ctrl+PageDown</kbd> | Next tab |
| <kbd>Ctrl+PageUp</kbd> | Previous tab |
| <kbd>Mod+=</kbd> | Zoom in |
| <kbd>Mod+-</kbd> | Zoom out |
| <kbd>Mod+0</kbd> | Reset zoom |
| <kbd>F1</kbd> | Open user manual |
| <kbd>Escape</kbd> | Close modal / cancel |

### Editor Shortcuts

Standard CodeMirror shortcuts work in the body editor and response viewer:

| Shortcut | Action |
|---|---|
| <kbd>Mod+Z</kbd> | Undo |
| <kbd>Mod+Shift+Z</kbd> | Redo |
| <kbd>Mod+A</kbd> | Select all |
| <kbd>Mod+F</kbd> | Find / search |

### Tab Bar

| Action | How |
|---|---|
| Close tab | Middle-click the tab |
| Context menu | Right-click a tab (Pin, Close, Close Others, Close All) |
